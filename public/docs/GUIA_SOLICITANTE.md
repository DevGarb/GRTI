## 1. Visão Geral

Como **Solicitante** no In Demands, seu papel é **registrar problemas e solicitações** para que a equipe técnica possa resolvê-los. Você tem acesso a:

- Abrir novos chamados de suporte (Software ou Hardware)
- Acompanhar o andamento dos seus chamados
- Receber notificações sobre atualizações de status
- Avaliar o atendimento após a resolução

### Menu disponível para o Solicitante
| Menu | Descrição |
|------|-----------|
| Dashboard | Painel com visão geral dos seus chamados |
| Chamados | Lista de chamados abertos por você |
| Configurações | Preferências pessoais |

---

## 2. Acesso e Login

1. Acesse a URL da plataforma fornecida pelo administrador
2. Insira seu **e-mail** e **senha**
3. Clique em **Entrar**

### Dicas
- Use o botão 🌙/☀️ no rodapé do menu para alternar entre **modo claro** e **modo escuro**
- Para sair, clique no botão de **logout** (ícone de porta) no rodapé do menu

---

## 3. Dashboard

Ao fazer login, você verá o **Dashboard** com um resumo dos seus chamados:

- **Contadores** por status (Aberto, Em Andamento, Aguardando Aprovação, etc.)
- **Gráficos** de evolução
- **Visão rápida** dos chamados mais recentes

---

## 4. Abrindo um Chamado

### Acessar: Menu → **Chamados** → botão **"+ Novo Chamado"**

### Passo a passo
1. Clique em **"+ Novo Chamado"**
2. Preencha os campos obrigatórios:
   - **Título**: Resumo curto e claro do problema (ex.: "Impressora não liga")
   - **Descrição**: Detalhes do problema — quanto mais informação, mais rápido o atendimento
3. Selecione:
   - **Prioridade**: Baixa, Média, Alta ou Urgente
   - **Tipo**: Software ou Hardware
4. (Opcional) Selecione um **técnico** para atribuir diretamente
5. (Opcional) Anexe **imagens** para ilustrar o problema:
   - Arraste e solte arquivos na área de upload
   - Ou use **Ctrl+V** para colar uma captura de tela
6. Clique em **"Criar Chamado"**

### Dicas para um bom chamado
| ✅ Faça | ❌ Evite |
|---------|---------|
| Descreva o problema de forma objetiva | Títulos vagos como "Não funciona" |
| Inclua capturas de tela | Abrir chamados duplicados |
| Informe o equipamento/sistema afetado | Deixar a descrição vazia |
| Selecione a prioridade correta | Marcar tudo como "Urgente" |

## 5. Acompanhando seus Chamados

### Acessar: Menu → **Chamados**

Na tela de chamados, você verá seus chamados agrupados por responsável. Cada chamado mostra:

- **Título** do chamado
- **Status** atual (com badge colorido)
- **Categoria** do serviço
- **Data** de criação
- **Prioridade**

### Fluxo de Status

```
Aberto → Em Andamento → Aguardando Aprovação → Aprovado → Fechado
```

| Status | O que significa |
|--------|----------------|
| **Aberto** | Chamado registrado, aguardando atendimento |
| **Em Andamento** | Técnico está trabalhando na solução |
| **Aguardando Aprovação** | Solução aplicada, aguardando sua confirmação |
| **Aprovado** | Você confirmou que o problema foi resolvido |
| **Fechado** | Chamado encerrado definitivamente |

### Visualizar detalhes
Clique em qualquer chamado para abrir o **modal de detalhes** com:
- Informações completas do chamado
- Nome do técnico responsável
- Anexos enviados
- Histórico de status

### Filtros
Use os filtros para encontrar chamados rapidamente:
- **Busca por texto**: Pesquise por título, descrição ou nome
- **Status**: Filtre por status específico
- **Período**: Selecione intervalo de datas

---

## 6. Avaliando o Atendimento

Após o técnico resolver seu chamado e alterar o status para **"Aguardando Aprovação"**:

1. Abra o chamado na lista
2. Verifique se o problema foi realmente resolvido
3. Se estiver satisfeito, o técnico fechará o chamado e poderá solicitar sua avaliação

### Avaliação
Quando solicitado, você pode avaliar o atendimento com:
- **Nota em estrelas** (1 a 5): Qualidade do atendimento
- **Comentário** (opcional): Observações sobre o serviço

Sua avaliação ajuda a melhorar a qualidade do suporte!

## 7. Configurações

### Acessar: Menu → **Configurações**

Ajuste suas preferências:
- **Tema**: Modo claro ou escuro
- **Informações do perfil**: Nome e dados pessoais

```
1. Login → Dashboard

4. Anexar imagens (opcional)
5. Criar Chamado ✅

```
